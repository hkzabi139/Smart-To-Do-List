## ✨ Key Features

* **📅 Dynamic Calendar Engine:** Har mahine ke dates aur weekdays ko automatically calculate aur display karta hai.
* **💾 Persistent Local Storage:** Saare tasks browser ki memory mein save hote hain, taake refresh karne par data loss na ho.
* **👁️ Conditional UI (Date-Specific):** Task add karne ka form tabhi nazar aata hai jab user kisi specific date par click karta hai.
* **📂 Task Filtering:** * **All:** Tamam tasks dikhata hai.
    * **Pending:** Sirf wo tasks jo abhi mukammal nahi hue.
    * **Done:** Wo tasks jo "Save/Check" ho chuke hain.
* **🎨 Pro UI/UX:** * **Responsive Grid:** Calendar aur Task panel desktop par barabar (equal) nazar aate hain aur mobile par stack ho jate hain.
    * **Priority Badges:** High, Medium, aur Low priority ke liye alag color indicators.
    * **Visual Indicators:** Calendar dates par "Dot" nazar aata hai agar us din koi task scheduled ho.

## 🛠️ How to Use

1.  **Select Date:** Calendar mein se kisi bhi date par click karein.
2.  **Add Task:** Right side par form khulega, wahan Title, Description, Priority aur Time enter karke "Add Task" par click karein.
3.  **Manage:**
    * ✔️ Button par click karke task ko "Done" mark karein.
    * 🗑️ Button se task permanent delete karein.
4.  **Filter:** Upar diye gaye buttons (All, Pending, Done) se apni list ko filter karein.
